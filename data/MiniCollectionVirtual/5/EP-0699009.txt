<INVENTION-TITLE>
Call mode sub-system for a mobile communications system with message service
</INVENTION-TITLE>
<CLASSIFICATIONS>H04W412	H04W414	H04W416	H04W416	</CLASSIFICATIONS>
<CLASSIFICATIONS-THIRD>H04W	H04W	H04W	H04W	</CLASSIFICATIONS-THIRD>
<CLASSIFICATIONS-FOURTH>H04W4	H04W4	H04W4	H04W4	</CLASSIFICATIONS-FOURTH>
<ABSTRACT>
The network sub-system (NSS) is provided for a mobile radio system (MRS) with one or more radio connection points (GMSC) for connecting a call from a calling subscriber (A) to a called subscriber (B) of the mobile radio system (MRS). A communication computer (COMP) is connected to the central connection point (SMSC). If the called subscriber cannot be reached, the computer generates a message (SM) containing one or more subscriber identification (A-ID) of the caller. A memory (MEM) is connected to a the computer for storing the message (SM) of the called subscriber. Preferably, the computer and the memory are both integrated in a short message service centre (SMSC). The computer and the memory (MEM) may generate and store the message as a short alphanumeric data sequence.
</ABSTRACT>
