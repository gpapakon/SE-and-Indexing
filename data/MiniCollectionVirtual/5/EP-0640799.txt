<INVENTION-TITLE>
Control unit for a heating installation.
</INVENTION-TITLE>
<CLASSIFICATIONS>F24D1900	F24D1910	G05D2319	G05D2319	</CLASSIFICATIONS>
<CLASSIFICATIONS-THIRD>F24D	F24D	G05D	G05D	</CLASSIFICATIONS-THIRD>
<CLASSIFICATIONS-FOURTH>F24D19	F24D19	G05D23	G05D23	</CLASSIFICATIONS-FOURTH>
<ABSTRACT>
Such a heating control unit is provided with an operating device (4) and a display unit (5a). The heating control unit communicates with other devices (1, 3, 3n) of the heating installation via a data bus (2). In the case of this control unit, the visual representation of the desired and actual values of the heating installation are to be improved and, in particular, clearly understandable information is to be displayed to the user in the case of adjustment operations and interrogations. For this purpose, a commercially available television set (5) is used as display unit (5a). In order to transmit the picture content, the television set (5) is connected directly by cable to the operating device (4) via its aerial (antenna) input or a standard SCART socket (receptacle).
