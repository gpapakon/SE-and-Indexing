<INVENTION-TITLE>
Method for characterizing and using the behaviour of users placed in front of multiple-choices and installation for using this method.
</INVENTION-TITLE>
<CLASSIFICATIONS>G06Q3000	G06Q3000	G07F700	G07F700	G07G100	G07G100	</CLASSIFICATIONS>
<CLASSIFICATIONS-THIRD>G06Q	G06Q	G07F	G07F	G07G	G07G	</CLASSIFICATIONS-THIRD>
<CLASSIFICATIONS-FOURTH>G06Q30	G06Q30	G07F7	G07F7	G07G1	G07G1	</CLASSIFICATIONS-FOURTH>
<ABSTRACT>
The method is characterized in that:   - Goods and/or services are selected and a reward aimed at the users is determined for each of them, - each user (U) is identified by coded data exploitable by computer means, - goods and/or services from those which were selected are designated, for each user considered individually, as a function of the identification of the said user, - each user is informed of, on the one hand, just the uncoded information (91-92) corresponding to the designated goods and/or services, and, on the other hand, of the declaration of the corresponding reward, printing them in situ, on a sheet (97-98) which each user (U) is able to retain and consult throughout his stay on the site,   - at the exit from the site, each user (U) is identified, - at the exit from the site, each good and/or service chosen on the site by each user is identified by means of coded data, and they are entered and stored by computer means, - each user having chosen at least one designated good and/or service is handed the corresponding reward.
