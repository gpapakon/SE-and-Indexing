<INVENTION-TITLE>
Electronic remote teaching system.
</INVENTION-TITLE>
<CLASSIFICATIONS>G09B500	G09B512	G09B700	G09B700	G09B706	</CLASSIFICATIONS>

<ABSTRACT>
The present invention relates to an electronic remote teaching system, in which each student terminal (50) is linked to a server centre (70) via the telephone network (30), this server centre being itself linked by this same telephone network to each teacher terminal (90), the student terminal including fax sending means (52, 58, 60) for addressing a document to be corrected to the teacher via the server centre, and fax-mode and character-mode receiving means (52, 60) for receiving the corrected document from the teacher, via the server centre, and the teacher terminal including interactive correction means (98) in order to allow direct correction of the document received from the student via the server centre by use of visual display means (96) of this terminal, without the requirement for fax printing means. A processing assembly (110) is linked to the server centre (70) can receive requests from the latter and is intended to deliver to it example corrected versions of these documents, addressed to the teacher by the student. Application to remote working.
