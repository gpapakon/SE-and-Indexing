<INVENTION-TITLE>
Electrical installation box for flush mounting.
</INVENTION-TITLE>
<CLASSIFICATIONS>F21V2102	F21V2102	H02G312	H02G312	H02G318	H02G320	</CLASSIFICATIONS>
<CLASSIFICATIONS-THIRD>F21V	F21V	H02G	H02G	H02G	H02G	</CLASSIFICATIONS-THIRD>
<CLASSIFICATIONS-FOURTH>F21V21	F21V21	H02G3	H02G3	H02G3	H02G3	</CLASSIFICATIONS-FOURTH>
<ABSTRACT>
An electrical installation box (10) is described, which is intended for flush mounting. The installation box (10) has a cylindrical wall (20) which is provided internally with a holding device (26) for detachable retention of a bridge element (28). The bridge element (28), which can expediently be fixed in an easily detachable manner to the installation box (10) in the manner of a bayonet fastening, has at least one female element (48) which is arranged on the bridge element (28) such that it can move linearly in the radial direction of the installation box (10).
