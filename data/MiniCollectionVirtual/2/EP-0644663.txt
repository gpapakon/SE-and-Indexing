<INVENTION-TITLE>
Method and apparatus for synchronisation in a direct sequence spread spectrum digital transmission system.
</INVENTION-TITLE>
<CLASSIFICATIONS>H04B1707	H04B1707	</CLASSIFICATIONS>
<CLASSIFICATIONS-THIRD>H04B	H04B	</CLASSIFICATIONS-THIRD>
<CLASSIFICATIONS-FOURTH>H04B1	H04B1	</CLASSIFICATIONS-FOURTH>
<ABSTRACT>
The invention consists, on transmission, in synchronising the clock of the pseudo-random code with the data timing clock, and in synchronising the data to the spreading code, the synchronisation of the data to the spreading code being performed for a position of the code corresponding to a predetermined binary state. Moreover, on transmission, "transparent" additional coding is introduced on one data bit every time the spreading code is in a position corresponding to a predetermined binary state. This additional coding makes it possible, on reception, to perform coherent demodulation without phase ambiguity.
