<INVENTION-TITLE>
Apparatus and method for guiding a vehicle

<ABSTRACT>
The device for guiding a vehicle to a destination has a radio transmitter and receiver and a satellite based position determining system in the vehicle. An arrangement is provided for inputting the vehicle destination. The destination together with the location determined by the position determining system can be transmitted via the transmitter and receiver device. On the basis of the transmitted information a proposed route to be stored in the device can be received. The proposed route contains the coordinates of base points lying on the route as well as base point instructions which contain additional information on a base point. The output os the base point instruction is controlled by the satellite position determination system. When a base point is reached a base point instruction which has not previously been output is output.
</ABSTRACT>
