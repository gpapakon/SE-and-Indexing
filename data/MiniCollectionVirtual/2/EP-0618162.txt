<INVENTION-TITLE>
Web splicing adhesive tape for a new web roll.
</INVENTION-TITLE>
<CLASSIFICATIONS>B65H1910	B65H1910	C09J702	C09J702	</CLASSIFICATIONS>
<CLASSIFICATIONS-THIRD>B65H	B65H	C09J	C09J	</CLASSIFICATIONS-THIRD>

<ABSTRACT>
The present invention relates to the automatic connection of a new web roll (10) to a running web (8) and provides a new web roll (10) which is equipped with adhesive connecting tapes (20, 100, 140, 190, 220). A highly adhesive double-sided holding or fastening adhesive tape (40, 80, 120, 170, 220) makes a strong connection between the new web (10) and the running web (8). A double-sided separating adhesive tape (20, 100, 140, 190, 250) is provided with an adhesive of high adhesion on one side and with an adhesive of low adhesion on the other side and fastens the first layer (16) on the new web roll (10) before connection. During the connection of the webs (8, 14), the separating adhesive tape (20, 100, 140, 190, 250) comes loose on its side of low adhesion, so that the new web roll (10) can unwind. The double-sided adhesive tapes are arranged either directly on the new web roll (10) or on a carrier strip (60, 110, 160, 230) which then covers the front edge (18) of the new web roll (10).
