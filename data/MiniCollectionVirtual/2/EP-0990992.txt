<INVENTION-TITLE>
DCT/IDCT processor and data processing method
</INVENTION-TITLE>
<CLASSIFICATIONS>G06F1714	G06F1714	</CLASSIFICATIONS>
<CLASSIFICATIONS-THIRD>G06F	G06F	</CLASSIFICATIONS-THIRD>
<CLASSIFICATIONS-FOURTH>G06F17	G06F17	</CLASSIFICATIONS-FOURTH>
<ABSTRACT>
The arrangement transforms a value (x) with N positions according to a determined transformation instruction (F) into a respective value (u) with M positions, whereby each of the N positions is occupied by a value taken from a determined first value range (A), and each of the M positions is occupied by a value taken from a determined second value range (B). The arrangement (1) comprises an input (11) for the supply of each value to be transformed. The associated transformed value is calculated according to the transformation instruction, and is provided at an output (12) of the arrangement. An additional arrangement (1') is provided for preventing an input of a predetermined value (x0) at the input of the transformation arrangement, and for calculating a corresponding transformed value (ug) in a manner, different from the calculation performed in the transformation arrangement.
</ABSTRACT>
