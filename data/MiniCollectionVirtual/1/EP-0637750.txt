<ABSTRACT>
The invention relates to a method for analysing sample liquids, in which data of high information density are applied to vessels having reagent solutions, which are used for carrying out and controlling the analysis. The invention furthermore relates to a method for producing reagent vessels provided with code carriers and a system for analysing sample liquids in which reagent vessels provided with code carriers are employed. A method according to the invention can preferably be employed in clinical analysis apparatuses, since reagent vessels provided with code carriers of high information density rationalise the analysis procedure.
