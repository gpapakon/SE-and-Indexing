<INVENTION-TITLE>
Method and system for the safe assignment of a password in an Internet environment
</INVENTION-TITLE>
<CLASSIFICATIONS>H04L2906	H04L2906	</CLASSIFICATIONS>
<CLASSIFICATIONS-THIRD>H04L	H04L	</CLASSIFICATIONS-THIRD>
<CLASSIFICATIONS-FOURTH>H04L29	H04L29	</CLASSIFICATIONS-FOURTH>
<ABSTRACT>
The method involves generating a new password that is newly combined with the user ID stored in a database, whereby the first half of the password is passed by the responsible employee of the provider to the user by telephone and further parts of the password are sent to the user by an automatically generated e-mail to the user's address.
</ABSTRACT>
