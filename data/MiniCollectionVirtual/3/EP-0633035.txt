Multichannel dosage system.

<ABSTRACT>
A multichannel dosage system (1) for dosage of preselected fluid flows from individual fluid flow sources (3), with a control unit (4) and with data fields which are located in the control unit and at least describe the dosage of the individual fluid flows and which can be retrieved on an operating surface (5), connected to the control unit, with a data input circuit (10) and a data output circuit (6), is to be improved in such a way that the data field allocated to a fluid flow source can be simply selected. In order to achieve this object the invention provides that selection switches (13) are allocated to individual fluid flow sources (3) as part of the data input circuit (10). These switches can be used to image at least segments of the data field belonging to the selected fluid flow source (3) on the operating surface (5) and by these switches at least parts of the data input circuit (10) can be brought into operative connection with the data field belonging to the selected fluid flow source (3).
