<INVENTION-TITLE>
Colour facsimile apparatus.
</INVENTION-TITLE>
<CLASSIFICATIONS>H04N100	H04N100	H04N146	H04N146	</CLASSIFICATIONS>

<ABSTRACT>
A colour facsimile device is to be implemented with commercially available inexpensive means. According to the invention, the colour facsimile device contains a personal computer (PC), to the standard data interface (V.24) of which a colour scanner (F) is connected. The personal computer (PC) also contains an ISDN adapter (TA) for transmitting data in transparent mode, and a printer interface (SCSI) for connecting a colour printer. The input of selective destinations and printing commands can be input by an input unit (E) on the menu control.
