<INVENTION-TITLE>
CONNECTION NETWORK AND METHOD FOR EXPANDING THE CONNECTION NETWORK.
</INVENTION-TITLE>
<CLASSIFICATIONS>H04Q364	H04Q368	</CLASSIFICATIONS>
<CLASSIFICATIONS-THIRD>H04Q	H04Q	</CLASSIFICATIONS-THIRD>
<CLASSIFICATIONS-FOURTH>H04Q3	H04Q3	</CLASSIFICATIONS-FOURTH>
<ABSTRACT>
The coupling bay has a number of coupling modules arranged in successive coupling stages (A, B), the coupling lines between them being provided via a permutation coupling structure supported in a frame (R). Each coupling module is connected to the permutation coupling structure via a respective interface (AEA, AEB). Only some of the interface devices associated with each coupling stage are used for connection of the coupling modules, the remainder being connected to corresponding interface devices of each other coupling stage.
</ABSTRACT>
