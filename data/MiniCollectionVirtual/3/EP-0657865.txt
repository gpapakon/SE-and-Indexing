<INVENTION-TITLE>
Method for transforming an n-bit color space to an m-bit color space.
</INVENTION-TITLE>
<CLASSIFICATIONS>G09G502	G09G502	H04N160	H04N160	H04N730	H04N730	H04N1104	H04N1104	</CLASSIFICATIONS>
<CLASSIFICATIONS-THIRD>G09G	G09G	H04N	H04N	H04N	H04N	H04N	H04N	</CLASSIFICATIONS-THIRD>
<CLASSIFICATIONS-FOURTH>G09G5	G09G5	H04N1	H04N1	H04N7	H04N7	H04N11	H04N11	</CLASSIFICATIONS-FOURTH>
<ABSTRACT>
The invention relates to a method for converting an n-bit colour image signal into an m-bit colour image signal which is as colour true as possible, in particular for driving an m-bit colour display (n 
>
 m). For this purpose, provision is made for a number of colours, the number corresponding to the bit number of the m-bit colour image signal, to be selected from a first colour space defined from the n-bit colour image signal, as a function of the frequency of the occurrence in the first colour space and for a colour deviation relative to a colour defined by the n-bit colour image signal to be reduced or compensated by selecting an adjacent colour of the selected colours.
</ABSTRACT>
