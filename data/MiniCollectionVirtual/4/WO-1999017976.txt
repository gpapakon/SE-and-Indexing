<INVENTION-TITLE>
CONTROLLER FOR AUTOMOBILE
</INVENTION-TITLE>
<CLASSIFICATIONS>B62D504	B62D504	G06F9445	G06F9445	</CLASSIFICATIONS>
<CLASSIFICATIONS-THIRD>B62D	B62D	G06F	G06F	</CLASSIFICATIONS-THIRD>
<CLASSIFICATIONS-FOURTH>B62D5	B62D5	G06F9	G06F9	</CLASSIFICATIONS-FOURTH>
<ABSTRACT>
A controller for automobile which is provided with a nonvolatile memory which is incorporated in a vehicle control computer and stores control software, and an auxiliary storage device which sets an unerasable area to the nonvolatile memory and rewrites the control software in an unerasable area in new control software.
</ABSTRACT>
