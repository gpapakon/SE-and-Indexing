<INVENTION-TITLE>
Interactive gambling game terminal.
</INVENTION-TITLE>

<ABSTRACT>
Games terminal comprising at least one video monitor (35) with tactile screen with which is associated a memory (20) intended to receive at least picture data, and a central unit (45) for processing games applications which may be selected on the tactile screen (12) and the corresponding operating programs, characterised in that the memory (20) is a single memory space for storing compressed data from pictures, from programs, from games applications, and in that it furthermore includes an access controller (21) intended to select the data stored in the memory space and to route them to means for processing the picture data (31, 32), program and applications data (47), associated respectively with the video monitor (35) and to means (33) for multiplexing the picture data, means (30) for real-time expansion being interposed between the access controller (21) and the picture data processing means (31, 32).
