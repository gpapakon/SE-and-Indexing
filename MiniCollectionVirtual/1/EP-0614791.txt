<INVENTION-TITLE>
Antitheft device for motor vehicle.
</INVENTION-TITLE>
<CLASSIFICATIONS>B60R2500	B60R2500	B60R2504	B60R2504	G07C900	G07C900	</CLASSIFICATIONS>
<CLASSIFICATIONS-THIRD>B60R	B60R	B60R	B60R	G07C	G07C	</CLASSIFICATIONS-THIRD>
<CLASSIFICATIONS-FOURTH>B60R25	B60R25	B60R25	B60R25	G07C9	G07C9	</CLASSIFICATIONS-FOURTH>
<ABSTRACT>
The antitheft device of the invention comprises a means (14, 15) for producing a confidential code, comprising an emission device (16) and at least one active receiver module (19, 20) arranged by the user at a location determined arbitrarily by himself or herself, the code production means and each module being connected to the existing electrical network of the car (2), each module comprising means (29 to 32) making it possible to recognise the validity of the code emitted by the code production means and transmitted by the electrical network (2), and means for acting on an essential element of the car.
