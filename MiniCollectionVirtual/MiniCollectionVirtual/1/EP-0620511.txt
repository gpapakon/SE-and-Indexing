<ABSTRACT>
The invention relates to an electrical device, especially a do-it-yourself electrical device, having an electronic control unit with which the use and the operation of the do-it-yourself electrical device is intended to be simplified by means of the incorporation of an electronic notebook. For this purpose it is provided that the electronic control unit has an input unit (10) connected to a memory unit (14) and an output unit (18) likewise connected to the memory unit (14) and that individual data and/or program for special use situations according to a predetermined algorithm can be stored in the memory unit (14).
