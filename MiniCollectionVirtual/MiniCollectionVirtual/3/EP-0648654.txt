<INVENTION-TITLE>
Security system for vehicles preventing unauthorised use.
</INVENTION-TITLE>
<CLASSIFICATIONS>B60R2504	B60R2504	</CLASSIFICATIONS>
<CLASSIFICATIONS-THIRD>B60R	B60R	</CLASSIFICATIONS-THIRD>
<CLASSIFICATIONS-FOURTH>B60R25	B60R25	</CLASSIFICATIONS-FOURTH>
<ABSTRACT>
A security system for vehicles, particularly motor vehicles, preventing unauthorised use is characterised by the utilization of a large-area centralised radio control system in the long wave frequency band and radio reception systems acting on the drive mechanism in the vehicles, which can be activated by radio addresses individually allocated to each vehicle. The radio reception system consists of a radio reception section, a generator and a coder. In the coder, an electronic key is stored which is initiated by the generator. This radio reception system is connected to the decoder of the drive control of the vehicle via an interface (SS).
