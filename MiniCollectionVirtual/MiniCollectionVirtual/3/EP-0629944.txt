<INVENTION-TITLE>
Data processing machine.
</INVENTION-TITLE>
<CLASSIFICATIONS>G06F938	G06F938	</CLASSIFICATIONS>
<CLASSIFICATIONS-THIRD>G06F	G06F	</CLASSIFICATIONS-THIRD>

<ABSTRACT>
The invention relates to a data processing machine, comprising a program counter, at least one data memory, a combinational logic unit and an accumulator. It is the object of the invention to reduce significantly the time requirement of this data processing machine. This object is achieved by means of two address look-up table modules, which address two data memories, the data of which are logically combined by means of a triggerable combinational logic unit.
</ABSTRACT>
