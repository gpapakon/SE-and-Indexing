<INVENTION-TITLE>
Method and device for controlling the smooth running of an internal combustion engine.
</INVENTION-TITLE>
<CLASSIFICATIONS>F02D4114	F02D4114	F02D4500	F02D4500	F02P5145	F02P5145	</CLASSIFICATIONS>
<CLASSIFICATIONS-THIRD>F02D	F02D	F02D	F02D	F02P	F02P	</CLASSIFICATIONS-THIRD>
<CLASSIFICATIONS-FOURTH>F02D41	F02D41	F02D45	F02D45	F02P5	F02P5	</CLASSIFICATIONS-FOURTH>
<ABSTRACT>
A method and a device are described for controlling the smoothness of running of an internal combustion engine. A segment is in each case defined by two segment pulses. A smooth running actual value, a smooth running set value, a control deviation and a control unit, which specifies a cylinder-specific adjustment value on the basis of the assigned control deviation, are assigned to each cylinder of the internal combustion engine. The smooth running actual values and the smooth running set values are specified on the basis of at least one segment period. The smooth running actual values and/or the control deviations fed to the control units are subjected to weighted averaging.
