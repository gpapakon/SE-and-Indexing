<INVENTION-TITLE>
Photo detector arrangement.
</INVENTION-TITLE>
<CLASSIFICATIONS>G01J110	G01J116	</CLASSIFICATIONS>
<CLASSIFICATIONS-THIRD>G01J	G01J	</CLASSIFICATIONS-THIRD>
<CLASSIFICATIONS-FOURTH>G01J1	G01J1	</CLASSIFICATIONS-FOURTH>
<ABSTRACT>
A description is given of an IR detector arrangement having photoconductive detector elements, in which a detector element which is screened off from incident radiation is used as reference element and the signal processing is carried out by forming the difference between the detector signals and the reference signal from the reference element.
