<INVENTION-TITLE>
Method of block coding an ATM cell using a product code
</INVENTION-TITLE>
<CLASSIFICATIONS>H04L100	H04L100	H04L1256	H04L1256	H04Q1104	H04Q1104	</CLASSIFICATIONS>
<CLASSIFICATIONS-THIRD>H04L	H04L	H04L	H04L	H04Q	H04Q	</CLASSIFICATIONS-THIRD>
<CLASSIFICATIONS-FOURTH>H04L1	H04L1	H04L12	H04L12	H04Q11	H04Q11	</CLASSIFICATIONS-FOURTH>
<ABSTRACT>
Two different codes are applied to the header and the data field of the ATM cell. The code used for the header is stronger and is a sub-code of the data field code. A third encoding is applied to the data obtained, in a direction perpendicular to the previous encoding, so that one word resulting from the third encoding contains a bit from each word of the two previous encodings. The encoding uses BCH codes from one of the following families: (9n, k), (n, k-1), (n+1, k), (n-s, k-s), (n-s, k-1-s) with integer k, n and s and s k. The data from the first two encoding passes is interlaced before the third encoding.
 
</ABSTRACT>
