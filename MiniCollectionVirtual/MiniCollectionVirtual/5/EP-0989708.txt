<INVENTION-TITLE>
System and usage independent multifunctional access module for digital and analog datatransmission of any type
</INVENTION-TITLE>
<CLASSIFICATIONS>H04L2906	H04L2906	H04N500	H04N500	</CLASSIFICATIONS>
<CLASSIFICATIONS-THIRD>H04L	H04L	H04N	H04N	</CLASSIFICATIONS-THIRD>
<CLASSIFICATIONS-FOURTH>H04L29	H04L29	H04N5	H04N5	</CLASSIFICATIONS-FOURTH>
<ABSTRACT>
The system has a replaceable multifunction access module that provides access to different data services and packets from different data sources. The data can be encoded differently, can be compressed, addressed or standardized in a non-uniform manner. and can be specified for any number of different applications. An Independent claim is also included for an application processing module for the system.
</ABSTRACT>
