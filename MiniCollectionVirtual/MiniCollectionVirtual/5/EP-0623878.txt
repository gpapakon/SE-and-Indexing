<INVENTION-TITLE>
Display device for installation bus system.
</INVENTION-TITLE>
<CLASSIFICATIONS>G06F1132	G06F1132	H02J1300	H02J1300	</CLASSIFICATIONS>
<CLASSIFICATIONS-THIRD>G06F	G06F	H02J	H02J	</CLASSIFICATIONS-THIRD>
<CLASSIFICATIONS-FOURTH>G06F11	G06F11	H02J13	H02J13	</CLASSIFICATIONS-FOURTH>
<ABSTRACT>
With the invention, a display device, which can be connected to an installation bus system, is proposed. The display device (6) contains multi-coloured LEDs (8), which can optionally be assigned to specified bus stations (2), under menu control from a PC, to display operating and fault states with different colours.
