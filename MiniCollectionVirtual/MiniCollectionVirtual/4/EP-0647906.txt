<INVENTION-TITLE>
Voice memory management method and apparatus.
</INVENTION-TITLE>
<CLASSIFICATIONS>G06F1202	G06F1202	H04M350	H04M3533	</CLASSIFICATIONS>
<CLASSIFICATIONS-THIRD>G06F	G06F	H04M	H04M	</CLASSIFICATIONS-THIRD>
<CLASSIFICATIONS-FOURTH>G06F12	G06F12	H04M3	H04M3	</CLASSIFICATIONS-FOURTH>
<ABSTRACT>
A method for the management of a voice memory (VMEM) and an apparatus therefor are presented. For access to the voice memory by a plurality of users (U1, U2, U3, U4), the said voice memory is subdivided into user memory areas (S1, S2, S3, S4) and a shared (multi-user) memory area (SALL). By adding a part (S4') of the shared memory area to one of the user memory areas (S4) or vice versa, the memory size of this user memory area is matched to the necessary storage requirement (Min4) of the user (U4).   The storage capacity of the voice memory VMEM is utilised effectively, since the users (U1 to U4) are allocated as far as possible no free memory location which they do not utilise. The apparatus can be used, for example, as a voice mailbox in a telecommunications system.
