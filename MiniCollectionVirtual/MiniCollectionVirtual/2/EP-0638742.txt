Control system for vehicles, in particular for automatic transmissions.

<ABSTRACT>
The control can serve in a motor vehicle for controlling an automatic transmission, a wheel slip control device, a rear wheel steering system, an active wheel suspension system etc. An adjustment variable (e.g. gear new) is specified for the device to be controlled as a function of the position of the accelerator pedal, of the speed of the vehicle and if appropriate of further sensor signals. The control has a fuzzy computer 23 to which the aforesaid measurement variables and parameters derived therefrom (activity of the accelerator pedal, average values of the accelerator pedal position or of the throttle valve position) are fed as input variables and by means of which an output variable (load, drive, gear-shift) is generated according to the rules of fuzzy logic with reference to evaluation of the input variables. By means of these output variables, the respective driving situation and load situation of the motor vehicle and the driving style of the driver are allowed for when specifying the adjustment variable.
