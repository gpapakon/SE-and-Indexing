<INVENTION-TITLE>
Software update for webphones
</INVENTION-TITLE>
<CLASSIFICATIONS>G06F9445	G06F9445	G06F2100	G06F2100	H04L2906	H04L2906	H04L2908	H04L2908	H04M1253	H04M1253	</CLASSIFICATIONS>
<CLASSIFICATIONS-THIRD>G06F	G06F	G06F	G06F	H04L	H04L	H04L	H04L	H04M	H04M	</CLASSIFICATIONS-THIRD>
<CLASSIFICATIONS-FOURTH>G06F9	G06F9	G06F21	G06F21	H04L29	H04L29	H04L29	H04L29	H04M1	H04M1	</CLASSIFICATIONS-FOURTH>
<ABSTRACT>
The updating method uses certified updating software which is loaded onto a web server connected to the web network with which the web peripheral is used, with downloading of the updating software only permitted when a verification check is completed. The certification of the updating software may be provided by the internet service provider using a public-key method, with verification after downloading before the software is loaded in the memory of the internet web peripheral.
</ABSTRACT>
